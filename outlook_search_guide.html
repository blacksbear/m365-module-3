Outlook Search | Quick Reference

Find any email in seconds using search operators, filters, and scope settings.

🔍 All Mailboxes Ctrl+E
Results will appear here as you type
Find by person
from:
Find emails sent by a specific person.
e.g. from:sarah or from:sarah@company.com
to:
Find emails you sent to a specific person.
e.g. to:michael
cc:
Find emails where someone was CC'd.
e.g. cc:manager@company.com
participants:
Find emails involving anyone - sender or recipient.
e.g. participants:david
Find by content
subject:
Search only the subject line.
e.g. subject:budget or subject:"Q1 report"
body:
Search only the email body text.
e.g. body:invoice
"exact phrase"
Wrap in quotes to search an exact phrase.
e.g. "please see attached"
category:
Find emails tagged with a colour category.
e.g. category:red or category:urgent
Find by date
received:
Filter by when the email arrived.
e.g. received:today · received:this week · received:last month
received:<date>
Emails received before a specific date.
e.g. received:<01/01/2026
received:>date
Emails received after a specific date.
e.g. received:>01/03/2026
sent:
Filter by when you sent an email.
e.g. sent:yesterday
Find by properties
hasattachment:yes
Emails that include file attachments.
combine: from:client hasattachment:yes
isflagged:yes
Emails you have flagged for follow-up.
e.g. isflagged:yes received:this week
isread:no
Only unread emails in results.
e.g. from:hr isread:no
folder:
Limit search to a specific folder.
e.g. folder:clients subject:invoice
Combine operators - power searching
from:sarah subject:budget hasattachment:yes from:client received:this week isread:no subject:"Q1 report" received:>01/01/2026 isflagged:yes from:manager folder:inbox hasattachment:yes received:last month

Good news The core search syntax - from:, to:, subject:, folder: - is identical between Gmail and Outlook. You already know more than you think.
Best for most searches
All Mailboxes
Searches your entire account - Inbox, all folders, Sent, Archive, and Deleted Items. Use this when you're not sure where the email is.
Precise results
Current Folder
Only searches the folder you currently have open. Useful when you know exactly where an email should be and want faster, focused results.
Advanced filter bar
Date range
Use the date picker in the filter bar to set a custom "from" and "to" date - no syntax needed.
Has attachment
Tick the "Has attachments" checkbox - quicker than typing hasattachment:yes.
Sort results
Sort by Date, From, Subject, or Size to scan large result sets quickly.
Focused Inbox
Search only covers Focused by default. Switch to All to also include Other items.

1
Start broad, then narrow down
Begin with one keyword. If you get too many results, add an operator. Layering one at a time is faster than a complex query from scratch.
budget → from:sarah budget → from:sarah subject:budget
2
Use quotes for exact phrases
Without quotes Outlook finds emails containing any of the words. Quotes force an exact phrase match - essential for locating specific text.
please see attached → "please see attached"
3
Partial names work in from: and to:
You don't need the full email address. First names, last names, or even partial names work - Outlook matches your contact directory and email history.
from:sarah · from:chen · from:sarah.c
4
Search Deleted Items to recover lost emails
Accidentally deleted emails stay in Deleted Items for 30 days. Use folder:"Deleted Items" to search there specifically.
folder:"Deleted Items" from:client subject:invoice
5
Find large emails to free up mailbox space
If your mailbox is getting full, search for large emails and sort results by size to find the biggest offenders quickly.
messagesize:>5000 hasattachment:yes
6
Natural language works too
Outlook understands plain English like "emails from last week" or "unread emails from Sarah" - you don't always need formal operators.
emails from sarah this week with attachments
